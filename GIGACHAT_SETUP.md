- ✅ **БЕСПЛАТНО** - 1000 запросов в месяц
- ✅ **Работает в РФ** - не нужен VPN
- ✅ **Отличный русский язык** - родная модель
- ✅ **Быстрые ответы** - 2-5 секунд
- ✅ **Хороший контекст** - до 32K токенов

## 📋 Пошаговая настройка

### 1. Регистрация на SberDevices

1. Зайди на https://developers.sber.ru/
2. Нажми **"Войти"** → войди через **Сбер ID** (можно через телефон)
3. Если нет Сбер ID — создай, это бесплатно

### 2. Создание проекта

1. Перейди в **"Профиль"** → **"Мои проекты"**
2. Нажми **"Создать проект"**
3. Дай название (например "RAG Books")
4. Заполни описание (любое)

### 3. Подключение GigaChat

1. В проекте перейди во вкладку **"GigaChat"**
2. Нажми **"Подключить"**
3. Принимай условия использования

### 4. Получение ключей

1. В разделе **"Ключи доступа"** нажми **"Создать ключ"**
2. Скопируй **Client ID** (выглядит как `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`)
3. Скопируй **Client Secret** (длинная строка)

⚠️ **Client Secret показывается только один раз!** Сохрани сразу!

### 5. Настройка в проекте

Создай файл `.env` в корне проекта:

```bash
# Файл: .env
GIGACHAT_CLIENT_ID=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
GIGACHAT_CLIENT_SECRET=твоя_секретная_строка
```

Или через переменные окружения:

```bash
# Windows PowerShell
$env:GIGACHAT_CLIENT_ID="xxx"
$env:GIGACHAT_CLIENT_SECRET="xxx"

# Linux/Mac
export GIGACHAT_CLIENT_ID="xxx"
export GIGACHAT_CLIENT_SECRET="xxx"
```

### 6. Проверка работы

```bash
python book_rag.py --interactive
```

Введи вопрос — должен прийти ответ от GigaChat!

## 🐛 Решение проблем

### "GigaChat auth error: 401"
- Проверь Client ID и Client Secret
- Убедись что проект активен
- Проверь что GigaChat подключен

### "GigaChat API error: 429"
- Превышен лимит запросов (1000/мес)
- Подожди следующего месяца

### "ModuleNotFoundError: No module named 'openai'"
```bash
pip install openai
```

### "GigaChat не отвечает"
- Проверь интернет соединение
- GigaChat работает только в РФ и некоторых странах СНГ
