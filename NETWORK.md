# 🌐 Настройка доступа из локальной сети

## Быстрый старт

### 1. Открой порт в брандмауэре

**Способ A: Через скрипт (от администратора)**

1. Нажми правой кнопкой на `open_firewall.bat`
2. Выбери **"Запуск от имени администратора"**

**Способ B: Вручную через PowerShell (от администратора)**

```powershell
netsh advfirewall firewall add rule name="Flask RAG Port 5000" dir=in action=allow protocol=TCP localport=5000
```

### 2. Узнай свой IP адрес

```bash
ipconfig
```

Найди строку **IPv4-адрес** (например: `192.168.1.34`)

### 3. Запусти сервер

```bash
python web_app.py
```

В консоли увидишь:
```
📍 Локальный доступ: http://localhost:5000
📍 Доступ из сети: http://192.168.1.34:5000
```

### 4. Подключись с другого устройства

**С телефона/планшета/другого ПК:**
- Открой браузер
- Введи: `http://192.168.1.34:5000`
- Готово! 🎉

## 🔒 Безопасность

### ⚠️ ВАЖНО:

Этот проект предназначен **ТОЛЬКО для локальной сети**!

**НЕ делай:**
- ❌ Не открывай порт 5000 в интернет
- ❌ Не используй проброс портов на роутере
- ❌ Не запускай на публичных WiFi без защиты

**Можно:**
- ✅ Домашняя WiFi сеть
- ✅ Офисная локальная сеть
- ✅ Виртуальная локальная сеть (VPN)

### Для продакшена:

Если нужен доступ из интернета:
1. Добавь аутентификацию (логин/пароль)
2. Используй HTTPS
3. Настрой reverse proxy (nginx)
4. Ограничь доступ по IP

## 🐛 Решение проблем

### "Не могу подключиться с другого устройства"

1. **Проверь брандмауэр:**
   ```bash
   netsh advfirewall firewall show rule name="Flask RAG Port 5000"
   ```

2. **Проверь что сервер запущен:**
   - В консоли должно быть: `Running on http://0.0.0.0:5000`

3. **Проверь IP адрес:**
   - Устройства должны быть в одной сети
   - Пример: `192.168.1.X` где X разное

4. **Попробуй отключить брандмауэр временно:**
   - Панель управления → Брандмауэр → Вкл/Выкл

### "Сервер запускается только на localhost"

Проверь в `web_app.py`:
```python
app.run(debug=False, host='0.0.0.0', port=5000)
```

Должно быть `host='0.0.0.0'` (доступ из сети)
НЕ `host='127.0.0.1'` (только локально)

### "Порт занят"

Измени порт в `web_app.py`:
```python
app.run(debug=False, host='0.0.0.0', port=5001)  # Другой порт
```

Не забудь открыть новый порт в брандмауэре!

## 📊 Информация

- **Порт по умолчанию:** 5000
- **Протокол:** HTTP (не HTTPS)
- **Доступ:** Только локальная сеть
- **Одновременных подключений:** до 10
