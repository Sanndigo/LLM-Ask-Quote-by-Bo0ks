# 📚 RAG - Поиск по книгам с AI-ответами

[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![License: Non-Commercial](https://img.shields.io/badge/license-NC--BY-red.svg)](LICENSE)
[![Version](https://img.shields.io/badge/version-v0.4.0-green.svg)](https://github.com/Sanndigo/LLM-Ask-Quote-by-Bo0ks)

Интеллектуальная система поиска и ответов на вопросы по текстовым документам с использованием **семантического чанкинга** и **Mistral AI API**.

## ✨ Возможности

### 🔍 Умный поиск с семантическим чанкингом
- **4764 чанка** с умными границами (от точки до точки)
- **Никогда не обрывает слова** - сохраняет целостность предложений
- **Семантическая группировка** - объединяет по смыслу
- Показывает 5 наиболее релевантных фрагментов
- **Отображает главы, строфы, части** если найдены

### 💬 Popup с контекстом
- **Кликни на результат** - откроется popup с контекстом
- **Показывает соседние чанки** (по 2 с каждой стороны)
- **Полный текст** без обрезки

### ❓ Ответы на вопросы с Mistral AI
- AI генерирует развернутые ответы на русском
- **Использует контекст из найденных фрагментов**
- **Не выдумывает** - отвечает только на основе контекста
- Показывает цитаты из книг

### 📖 Управление библиотекой
- Веб-интерфейс с современным дизайном
- Загрузка TXT файлов
- Автоматическая индексация новых книг
- Поддержка русских и английских текстов

## ❗ Важно

- Задавайте **развернутые вопросы** - так ИИ лучше понимает контекст
- Система отвечает **ТОЛЬКО на основе загруженных книг**
- Если книги нет - загрузите её в папку `data/`

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка Mistral AI API

**Получи ключ:**
1. Зайди на https://console.mistral.ai/
2. Зарегистрируйся
3. Перейди в **API Keys**
4. Создай ключ

**Настрой .env:**
```bash
# Создай файл .env в корне проекта
echo MISTRAL_API_KEY=твой_ключ > .env
```

**Бесплатно:** 30 дней trial, потом ~$0.15 за 1M токенов

### 3. Индексация книг

```bash
# Пересоздать индекс (если изменились книги)
python main_processor.py --step all --threshold 0.45
```

### 4. Запуск веб-интерфейса

```bash
python web_app.py
```

Откройте в браузере: **http://localhost:5000**

### 5. Консольный режим (опционально)

```bash
python book_rag.py
```

```
├── data/                   # TXT книги
├── processed/              # Обработанные чанки (не коммитить)
└── embeddings/             # Векторный индекс (не коммитить)
```

## ⚙️ Настройка

### Переменные окружения (.env)

```bash
# Mistral AI API
MISTRAL_API_KEY=твой_ключ
```

### Параметры обработки книг

```bash
# Обработка с кастомными параметрами
python main_processor.py --step all --threshold 0.45
```

| Параметр | Значение | Описание |
|----------|----------|----------|
| `--threshold` | 0.45 | Порог схожести для чанкинга (0-1) |
| `--step` | all | Этап: process/embed/all |

## 🐛 Решение проблем

### "Mistral API error: 401"
- Проверь API ключ в `.env`
- Убедись что ключ активен
- Проверь что trial период не истек

### "Mistral API error: 429"
- Превышен лимит запросов
- Подожди или обнови тариф

### "ModuleNotFoundError: No module named 'openai'"
```bash
pip install openai
```

### Popup не открывается
- **Обнови страницу (F5)**
- Открой консоль браузера (F12) → проверь ошибки
- Убедись что JavaScript включен

### Чанки обрываются на полуслове
- **Пересоздай индекс:**
  ```bash
  rmdir /s /q processed embeddings
  python main_processor.py --step all
  ```
